Aprenda los Fundamentos de la Base de Datos MySQL. Este módulo cubre la Instalación del Servidor de Base de Datos MariaDB, Conexión a la Base de Datos y Shell SQL, Gestión de Bases de Datos y Usuarios, Copia de Seguridad y Restauración. Dominará estas habilidades esenciales de Linux a través de laboratorios prácticos y desafíos del mundo real.
